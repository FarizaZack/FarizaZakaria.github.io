# Fariza Zakaria — Portfolio

A modern, single-page personal portfolio positioning me at the intersection of
**finance, data, and AI automation**.

🔗 **Live site:** https://farizazack.github.io/

## Sections
- **Hero** — identity and focus
- **About** — background (finance/accounting + data + automation)
- **Featured Projects** — SQL analytics, R fraud analysis, Tableau dashboards
- **Skills & Tools**
- **Finance · Data · AI Focus**
- **Contact**

## Built with
- Plain HTML, CSS, and a little vanilla JavaScript — no framework, no build step
- Fully responsive, dependency-free (inline SVG icons)
- Hosted on GitHub Pages

## Local preview
Open `index.html` in any browser, or serve the folder:

```bash
python -m http.server 8080
# then visit http://localhost:8080
```
